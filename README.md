# 😄hAppy

## 📖 Sobre o Projeto

Este projeto foi desenvolvido a partir de um protótipo criado no Figma e tem como objetivo transformar um design de interface em uma aplicação web funcional utilizando os componentes e recursos do Bootstrap.

O sistema é composto por duas páginas principais:

* **Home:** apresentação inicial do sistema e acesso ao formulário.
* **Dados:** formulário para cadastro de informações do estabelecimento.

Todo o layout foi construído utilizando exclusivamente os recursos do **Bootstrap**, sem a criação de estilos personalizados em CSS, explorando o sistema de grid, componentes e classes utilitárias oferecidas pelo framework.

---

## 🚀 Funcionalidades

### 🏠 Página Home

* Tela inicial baseada em protótipo do Figma.
* Apresentação visual do sistema.
* Botão de navegação para a página de cadastro.
* Estrutura responsiva utilizando Grid System do Bootstrap.

### 📝 Página Dados

* Formulário para cadastro de estabelecimentos.
* Campos para informações comerciais.
* Seleção de categorias.
* Cadastro de endereço.
* Cadastro de telefone e WhatsApp.
* Opções de atendimento.
* Área para inserção de fotos.
* Organização dos campos utilizando componentes Bootstrap.
